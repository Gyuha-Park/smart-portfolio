# 🤖 AI 챗봇 위젯 UI 명세서

이 문서는 Smart Portfolio의 핵심 기능인 AI 챗봇 위젯에 대한 상세 UI 명세입니다.

## 1. 레이아웃 구조 (Layout Structure)

### 1.1 위치 및 배치 (Positioning)
- **위치**: 모든 페이지(`Global Layout`)의 **우측 하단(Bottom-Right)** 고정.
- **Z-Index**: 최상위 (다른 요소 위에 뜸).
- **여백 (Margin)**: 화면 끝에서 약 `20px ~ 30px` 떨어짐.

### 1.2 상태별 UI (States)

#### A. 닫힌 상태 (Collapsed)
- **형태**: 원형 플로팅 버튼 (Floating Action Button).
- **크기**: `60px x 60px`.
- **아이콘**: 챗봇/메시지 아이콘 (중앙 정렬).
- **스타일**: Primary Color 배경, 그림자 효과(Drop Shadow) 적용.
- **인터랙션**:
    - **Hover**: 살짝 커짐 (Scale 1.1) + 툴팁("AI에게 물어보세요").
    - **Click**: 채팅창 열림 (Expand Animation).

#### B. 열린 상태 (Expanded)
- **형태**: 카드 형태의 채팅 윈도우.
- **크기**:
    - **Desktop**: 너비 `380px ~ 400px`, 높이 `500px ~ 600px`.
    - **Mobile**: 화면 전체 (`100vw`, `100vh`) 또는 하단 시트 높이 `80vh` 이상.
- **구성 요소**:
    1.  **헤더 (Header)**:
        - **타이틀**: "포트폴리오 AI 어시스턴트" (Bold).
        - **상태 표시**: '온라인' 초록 점(Indicator).
        - **닫기/최소화 버튼**: 우측 상단 `X` 또는 `∨` 아이콘.
    2.  **대화 영역 (Chat Body)**:
        - 스크롤 가능한 메시지 리스트 영역.
        - 배경색: 아주 연한 회색 (구분감).
    3.  **입력 영역 (Input Area)**:
        - **텍스트 필드**: 멀티라인 입력 가능 (Auto-grow), Placeholder("궁금한 점을 입력해주세요...").
        - **전송 버튼**: 입력 내용이 있을 때만 활성화(Color change).

## 2. 상세 기능 및 인터랙션 (Interactions)

### 2.1 초기 진입 (Initial State)
- 채팅창을 처음 열면, AI의 **환영 메시지**가 먼저 표시됨.
- **메시지 내용**:
  > "안녕하세요! 👋\n포트폴리오에 대해 궁금한 것이 있으시면 편하게 질문해주세요.\n(예: 이 프로젝트 기술 스택이 뭐야?)"

### 2.2 로딩 상태 (Loading)
- 사용자가 메시지를 전송한 직후, AI 응답이 오기 전까지 표시.
- **UI**: 상대방 말풍선 위치에 **점 3개가 물결치는 애니메이션 (Typing Indicator)** 표시.
- **UX**: "AI가 답변을 생각 중입니다..." 라는 느낌 전달.

### 2.3 에러 및 예외 처리 (Error Handling)
- **네트워크 에러**: "연결이 불안정합니다. 다시 시도해주세요." (Toast 또는 시스템 메시지).
- **무관한 질문 (Off-topic)**:
  - 사용자가 포트폴리오와 관련 없는 질문(예: "오늘 점심 뭐 먹지?")을 했을 때.
  - **AI 답변**: "죄송합니다. 😅 저는 포트폴리오와 관련된 질문에만 답변드릴 수 있습니다. 프로젝트나 경력에 대해 물어봐 주세요!"

### 2.4 컨텍스트 연동 (Context Awareness)
- **상세 페이지 진입 시**:
  - 사용자가 `/project/ecommerce-app` 페이지를 보고 있다면, 챗봇을 열었을 때 자동으로 해당 프로젝트 컨텍스트가 주입됨.
  - **추천 질문 칩**: 채팅창 하단에 "이 프로젝트 기간은?", "사용 기술은?" 같은 칩(Chip) 버튼을 띄워 원클릭 질문 유도.

### 2.5 리드 수집 트리거 (Lead Capture)
- 대화 중 이메일/전화번호 패턴이 감지되거나, AI가 "견적 상담을 위해 연락처를 남겨주시겠어요?"라고 물었을 때.
- **폼(Form) UI**: 말풍선 대신 간단한 **입력 폼(이메일 입력창 + 제출 버튼)**이 대화창 내에 인라인으로 렌더링될 수 있음.

## 3. 디자인 스타일 가이드 (Style Guide)

- **컬러**:
    - **Primary**: 브랜드 메인 컬러 (버튼, 내 말풍선 배경).
    - **Secondary**: 챗봇 말풍선 배경 (연한 회색).
    - **Text**: 가독성 높은 짙은 회색 (`#333`).
- **타이포그래피**:
    - **본문**: 14px ~ 16px.
    - **헤더**: 16px ~ 18px Bold.
- **모션 (Motion)**:
    - 열리고 닫힐 때 부드러운 `Spring` 또는 `Ease-out` 애니메이션 적용.
